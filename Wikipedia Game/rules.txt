RULES:
	* start page and end page must be normal wikipedia articles in the same language
	* all pages must be within en.wikipedia.org or simple.wikipedia.org
	* no main page, special page, or random page visits
	* no searches
	* in page search is allowed
TIMED:
	* fastest player wins
	* stopwatch is started on first article and ended upon reaching last article
STEPS:
	* player with the fewest steps wins
	* every page except the first counts as one page visit
	* disambiguation pages count as page visits
	*category page sidebar links count as one page (even if a table must be unhidden to reveal the link)
